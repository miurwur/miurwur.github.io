Лабораторная работа №2

Цель работы

Изучение основных возможностей библиотеки NumPy для численных вычислений и анализа данных

Задачи

  1. Освоить создание и обработку массивов в NumPy
  2. Научиться выполнять векторные и матричные операции
  3. Провести статистический анализ данных
  4. Визуализировать результаты с помощью matplotlib и seaborn

Структура проекта:

main.py - основной файл с функциями
test.py - тесты
plots - графики

Реализованные функции

1. Создание и обработка массивов

  • create_vector() - создание вектора от 0 до 9 с использованием np.arange()
  • create_matrix() - генерация случайной матрицы 5x5 через np.random.rand()
  • reshape_vector() - изменение формы массива с помощью reshape()
  • transpose_matrix() - транспонирование матриц

2. Векторные операции

  • vector_add() - сложение двух векторов
  • scalar_multiply() - умножение вектора на скаляр
  • elementwise_multiply() - поэлементное умножение двух массивов
  • dot_product() - вычисление скалярного произведения векторов с помощью np.dot()

3. Матричные операции

  • matrix_multiply() - умножение матриц с использованием оператора @ или np.matmul()
  • matrix_determinant() - вычисление определителя квадратной матрицы через np.linalg.det()
  • matrix_inverse() - нахождение обратной матрицы с помощью np.linalg.inv()
  • solve_linear_system() - решение системы линейных уравнений Ax = b через np.linalg.solve()

4. Статистический анализ

  • load_dataset() - загрузка данных из CSV-файла с помощью pd.read_csv() и преобразование в NumPy массив через .to_numpy()
  • statistical_analysis() - вычисление статистических показателей (среднее, медиана, стандартное отклонение, минимум, максимум, 25й и 75й перцентили) с использованием np.mean(), np.median(), np.std(), np.min(), np.max(), np.percentile()
  • normalize_data() - Min-Max нормализация данных в диапазон [0, 1] по формуле (x - min) / (max - min)

5. Визуализация

  • plot_histogram() - построение гистограммы распределения оценок по математике с использованием plt.hist() и сохранение в файл math_histogram.png
  • plot_heatmap() - создание тепловой карты корреляции между предметами через sns.heatmap() с аннотациями и сохранение в correlation_heatmap.png
  • plot_line() - построение линейного графика зависимости оценки от номера студента с помощью plt.plot() и сохранение в student_scores_line.png
